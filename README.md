# StudyFlow

StudyFlow — это персональный ассистент для студентов, который помогает структурировать учебный процесс. Приложение позволяет группировать задания по дисциплинам, отслеживать сроки выполнения и визуализировать общий прогресс в учебе.

## Текущий функционал
* **Менеджмент дисциплин:** просмотр списка изучаемых предметов с наглядной индикацией прогресса выполнения заданий по каждому из них.
* **Управление задачами:** детальный просмотр списка заданий (лабораторные, тесты, отчеты), привязанных к конкретной дисциплине.
* **Интерфейс:** легкий дизайн на базе Material 3.

## Технологический стек
* **Язык:** Kotlin
* **UI:** XML, RecyclerView
* **Navigation:** Jetpack Navigation Component (NavHost, Fragments, Safe Args)
* **Архитектура:** Начальный этап внедрения MVVM
* **Сборка:** Gradle (Kotlin DSL)


## В перспективе:
- **Локальное хранилище:** использование Room Database для сохранения данных между сессиями.
- **Контроль дедлайнов:** система цветовой индикации срочности задач и уведомления.
- **Фильтрация:** возможность сортировки задач по статусу выполнения и приоритету.
